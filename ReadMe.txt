Данное приложение позволяет: 
- Вводить фигуру-примитив одного из четырех типов
- Выполнять над фигурами пространственно-геометрические операции
- Выполнять над фигурами теоретико-множественные операции

В сумме эти функции дают прототип простенького графического редактора-рисовалки.
